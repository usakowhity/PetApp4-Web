# 🐾 **PetApp4-Web（Myu Edition） — 最新版 README（日本語）**

PetApp4-Web（Myu Edition）は、静止画ベースの Web ペットアプリです。  
猫は **瞬き検出（EAR）**、**音声解析（声質分類＋長音検出）**、  
さらに **スワイプ／マウス操作** に反応して、さまざまな状態に変化します。

また、ユーザーの猫好き度合を判定する **猫語検定機能（世界初／Joke要素あり）** を搭載しています。

## 🐾 主な機能

### 1. 瞬き検出（EAR）
ゆっくり瞬きをすると **Approach（近づく）** 状態になります。  
さらに、連続したゆっくり瞬きで **Attention → Affection（甘える）** へと遷移します。

---

### 2. 音声トリガー（声質分類＋長音検出）

#### ■ 声質分類（従来機能）
- **柔らかい声** → Attention（注目）  
- **猫なで声（優しい声）** → Affection（甘える）  
- **荒い声** → Avoidance（避ける）

#### ■ 長音検出（新機能）
猫の鳴き声に特徴的な「ニャー」「ミャー」「ニャーオ」「ニャーゴ」などを検出します。

| 検出名 | 実際の鳴き声 | 状態遷移 |
|--------|--------------|----------|
| **nyan_short** | ニャッ | Attention |
| **nyan_long** | ニャー | Affection |
| **mya_long** | ミャー | Affection |
| **nyao** | ニャーオ | Play |
| **nyago** | ニャーゴ | Play |

長音は **周波数（mid/high）＋音量＋持続時間（300〜900ms）** で判定しています。

---

### 3. スワイプ／マウス操作
- ゆっくりスワイプ → **Affection（甘える）**  
- 早いスワイプ → **Play（遊ぶ）**  
- マウス高速移動 → **Play（遊ぶ）**  
- 画面タップ → **Avoidance（避ける）**

---

### 4. 自然な睡眠遷移
10秒間無操作で **Sleep（眠る）** に入り、  
その後自動で以下の流れになります：

```
Sleep（7秒）
↓
Stretch（2秒）
↓
Neutral（通常）
```

---

### 5. 猫語検定（世界初／Joke要素）
ユーザーの行動（瞬き／声／猫語／スワイプ）に応じて  
**猫語検定ランク（1〜5級）** が上下します。

- 甘える行動 → ランク上昇  
- 避ける行動 → ランク下降  

ランクは画面上部に **大きな肉球印🐾** とともに表示されます。
